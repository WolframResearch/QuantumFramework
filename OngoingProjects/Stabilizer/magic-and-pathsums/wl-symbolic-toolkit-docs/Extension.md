# Extension | [SpanFromLeft]

> [Extension](https://reference.wolfram.com/language/ref/Extension.html) — is an option for various polynomial and algebraic functions that specifies generators for the algebraic number field to be used.

## Details

For polynomial functions, [Extension](https://reference.wolfram.com/language/ref/Extension.html) determines the algebraic number field in which the coefficients are assumed to lie.

The setting [Extension](https://reference.wolfram.com/language/ref/Extension.html)->*a* specifies the field $\mathbb{Q}[a]$ consisting of the rationals extended by the algebraic number `*a*`.

[Extension](https://reference.wolfram.com/language/ref/Extension.html)->{*a*_1,*a*_2,…} specifies the field $\mathbb{Q}[a_{1},a_{2},\ldots]$.

The `*a*_*i*` must be exact numbers, and can involve radicals as well as [Root](https://reference.wolfram.com/language/ref/Root.html) and [AlgebraicNumber](https://reference.wolfram.com/language/ref/AlgebraicNumber.html) objects.

[Extension](https://reference.wolfram.com/language/ref/Extension.html)->[Automatic](https://reference.wolfram.com/language/ref/Automatic.html) specifies that any algebraic numbers that appear in the input should be included in the extension field.

For polynomial functions, the default setting [Extension](https://reference.wolfram.com/language/ref/Extension.html)->[None](https://reference.wolfram.com/language/ref/None.html) specifies that all coefficients are required to be rational. Any algebraic numbers appearing in input are treated like independent variables.

[Extension](https://reference.wolfram.com/language/ref/Extension.html)->{*a*_1,*a*_2,…} includes both the `*a*_*i*` and any algebraic numbers in the input.

[GaussianIntegers](https://reference.wolfram.com/language/ref/GaussianIntegers.html)->[True](https://reference.wolfram.com/language/ref/True.html) is equivalent to [Extension](https://reference.wolfram.com/language/ref/Extension.html)->[I](https://reference.wolfram.com/language/ref/I.html).

## Examples

### Basic Examples

[Factor](https://reference.wolfram.com/language/ref/Factor.html) a polynomial over $\mathbb{Q}[\sqrt{2}]$:

```wolfram
Factor[x^2-2,Extension->Sqrt[2]]
(* Output *)
-(Sqrt[2]-x) (Sqrt[2]+x)
```

[PolynomialGCD](https://reference.wolfram.com/language/ref/PolynomialGCD.html) over the field generated by the algebraic numbers present in the coefficients:

```wolfram
PolynomialGCD[x^2-2Sqrt[3]x+3,x^2-3,Extension->Automatic]
(* Output *)
Sqrt[3]-x
```

### Scope

By default, factorization is performed over the rationals:

```wolfram
Factor[x^4-9]
(* Output *)
(-3+x^2) (3+x^2)
```

This specifies the factorization should be done over the rationals extended by `Sqrt(3)`:

```wolfram
Factor[x^4-9,Extension->Sqrt[3]]
(* Output *)
-(Sqrt[3]-x) (Sqrt[3]+x) (3+x^2)
```

Here the factorization is done over the rationals extended by `Sqrt(3)`and [I](https://reference.wolfram.com/language/ref/I.html):

```wolfram
Factor[x^4-9,Extension->{Sqrt[3],I}]
(* Output *)
-(Sqrt[3]-x) (Sqrt[3]-ⅈ x) (Sqrt[3]+ⅈ x) (Sqrt[3]+x)
```

By default, [PolynomialGCD](https://reference.wolfram.com/language/ref/PolynomialGCD.html) treats algebraic numbers as independent variables:

```wolfram
PolynomialGCD[x^3-x+1,x^2-Root[#^3-#+1&,1]^2]
(* Output *)
1
```

This computes the GCD over the algebraic number field generated by the coefficients:

```wolfram
PolynomialGCD[x^3-x+1,x^2-Root[#^3-#+1&,1]^2,Extension->Automatic]
(* Output *)
x-Root
```

By default, [Together](https://reference.wolfram.com/language/ref/Together.html) treats algebraic numbers as independent variables:

```wolfram
Together[1/(Sqrt[2]+x)+1/(x^2-2)]
(* Output *)
(-2+Sqrt[2]+x+x^2)/((Sqrt[2]+x) (-2+x^2))
```

With [Extension](https://reference.wolfram.com/language/ref/Extension.html)->[Automatic](https://reference.wolfram.com/language/ref/Automatic.html), [Together](https://reference.wolfram.com/language/ref/Together.html) recognizes algebraically dependent coefficients:

```wolfram
Together[1/(Sqrt[2]+x)+1/(x^2-2),Extension->Automatic]
(* Output *)
(-1+Sqrt[2]-x)/((Sqrt[2]-x) (Sqrt[2]+x))
```

By default, the norm is computed in the field generated by the [AlgebraicNumber](https://reference.wolfram.com/language/ref/AlgebraicNumber.html) object:

```wolfram
a=ToNumberField[Sqrt[2],Sqrt[2]+Sqrt[3]]
(* Output *)
AlgebraicNumber[Root,{0,-(9)/(2),0,(1)/(2)}]
```

```wolfram
AlgebraicNumberNorm[a]
(* Output *)
-2
```

This computes the norm in the field in which the [AlgebraicNumber](https://reference.wolfram.com/language/ref/AlgebraicNumber.html) object is represented:

```wolfram
AlgebraicNumberNorm[a,Extension->Automatic]
(* Output *)
4
```

This computes the norm in the field generated by $\sqrt{2}+\sqrt{3}+\sqrt{5}$:

```wolfram
AlgebraicNumberNorm[a,Extension->Sqrt[2]+Sqrt[3]+Sqrt[5]]
(* Output *)
16
```

### Properties & Relations

For [Factor](https://reference.wolfram.com/language/ref/Factor.html), [Extension](https://reference.wolfram.com/language/ref/Extension.html)->[I](https://reference.wolfram.com/language/ref/I.html) is equivalent to [GaussianIntegers](https://reference.wolfram.com/language/ref/GaussianIntegers.html)->[True](https://reference.wolfram.com/language/ref/True.html):

```wolfram
Factor[x^4+1,Extension->I]
(* Output *)
(-ⅈ+x^2) (ⅈ+x^2)
```

```wolfram
Factor[x^4+1,GaussianIntegers->True]
(* Output *)
(-ⅈ+x^2) (ⅈ+x^2)
```
