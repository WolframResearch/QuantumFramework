# CylindricalDecompositionFunction | [SpanFromLeft]

> [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html)[*data*][*x*_1,*x*_2,…]  — represents a cylindrical algebraic formula in `*x*_1,*x*_2,…`.

## Details

[CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html)[…][*x*_1,*x*_2,…] is generated by [CylindricalDecomposition](https://reference.wolfram.com/language/ref/CylindricalDecomposition.html)[*expr*, {*x*_1,*x*_2,…},"Function"].

[CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) objects provide an explicit compact representation of semialgebraic sets that can be efficiently used in further computation.

[CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) is typically used for repeated computations with semialgebraic sets, including computing Boolean combinations of sets, restricting sets with additional conditions, eliminating some variables or optimizing over sets.

A cylindrical algebraic formula in `*x*_1,…,*x*_*n*` has the form $F=C_{1}\vee \ldots \vee C_{k}$, where $C_{i}=B_{i,1}(x_{1})\wedge B_{i,2}(x_{1},x_{2})\ldots \wedge B_{i,n}(x_{1},\ldots,x_{n})$. Each $B_{i,j}$ has the form either $x_{j}=r_{i,j}(x_{1},\ldots,x_{j-1})$ or $r_{i,j}(x_{1},\ldots,x_{j-1})<x_{j}<s_{i,j}(x_{1},\ldots,x_{j-1})$, where $r_{i,j}$ and $s_{i,j}$ are algebraic functions that are defined and continuous on the solution set of $B_{i,1}\wedge \ldots \wedge B_{i,j-1}$. The solution set $Z(C_{i})$ of $C_{i}$ in $\mathbb{R}^{n}$ is called a cell. Projections of cells $Z(C_{i_{1}})$ and $Z(C_{i_{2}})$ on $\mathbb{R}^{m}$, for any $1 \leq m \leq n$, are either disjoint or identical.

[CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) provides an encapsulated representation of cylindrical algebraic formulas, which is often more efficient when used in the input to [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) or [CylindricalDecomposition](https://reference.wolfram.com/language/ref/CylindricalDecomposition.html), or to solvers such as [Reduce](https://reference.wolfram.com/language/ref/Reduce.html), [Resolve](https://reference.wolfram.com/language/ref/Resolve.html), [FindInstance](https://reference.wolfram.com/language/ref/FindInstance.html), [Solve](https://reference.wolfram.com/language/ref/Solve.html) or [Minimize](https://reference.wolfram.com/language/ref/Minimize.html).

[Normal](https://reference.wolfram.com/language/ref/Normal.html) converts [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) objects to explicit Boolean combinations of equations and inequalities.

## Examples

### Basic Examples

Find a cylindrical algebraic formula representation of the open unit ball:

```wolfram
CylindricalDecomposition[x^2+y^2+z^2<1,{x,y,z},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y,z]
```

Represent the single-cell solution as explicit inequalities:

```wolfram
Normal[%]
(* Output *)
-1<x<1&&-Sqrt[1-x^2]<y<Sqrt[1-x^2]&&-Sqrt[1-x^2-y^2]<z<Sqrt[1-x^2-y^2]
```

Find a cylindrical algebraic formula representation of the solution set of a quantified formula:

```wolfram
f=CylindricalDecomposition[Exists[z,x^2+y^2+z^2<=3&&z^3-2 x y z==2],{x,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y]
```

Compute the boundary of the solution set:

```wolfram
CylindricalDecomposition[f,{x,y},"BoundaryFunction"]
(* Output *)

```

Visualize the result:

Solve an equation over the solution set:

```wolfram
CylindricalDecomposition[f&&x^3+y^3==x y,{x,y},"Function"]
```

### Scope

#### Basic Uses

Solution of polynomial equations and inequalities:

```wolfram
CylindricalDecomposition[x^2+y^2<=x y z&&z^2-x y==3,{x,y,z},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y,z]
```

Coefficients can include real algebraic numbers:

```wolfram
CylindricalDecomposition[Sqrt[2]x^2+Root[#^5-2#-11&,1]y^2<1,{x,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y]
```

Coefficients can include real exact transcendental numbers:

```wolfram
CylindricalDecomposition[E^Pi x^2+Pi^E y^2<=77&&Log[2]x+Log[3]y==2,{x,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y]
```

Solution of a quantified system of polynomial equations and inequalities:

```wolfram
CylindricalDecomposition[Exists[z,x^2+y^2<=x y z&&z^2-x y==3],{x,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y]
```

[CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) with a constant truth value simplifies to [True](https://reference.wolfram.com/language/ref/True.html) or [False](https://reference.wolfram.com/language/ref/False.html):

```wolfram
CylindricalDecomposition[ForAll[z,x^2+y^2<=x y z&&z^2-x y==3],{x,y},"Function"]
(* Output *)
False
```

[CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) with real numeric initial arguments simplifies automatically:

```wolfram
CylindricalDecomposition[x^4+y^4<=z&&z^3-x y==1,{x,y,z},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y,z]
```

```wolfram
%/.x->1/2
(* Output *)
CylindricalDecompositionFunction[...][y,z]
```

```wolfram
%/.y->-2/3
(* Output *)
CylindricalDecompositionFunction[...][z]
```

```wolfram
Normal[%]
(* Output *)
z==Root
```

Use [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) objects in further computation:

```wolfram
f=CylindricalDecomposition[x^2+y^2<=3&&x^3-y^2>=1,{x,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y]
```

Compose a [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) with an algebraic mapping:

```wolfram
g=CylindricalDecomposition[Head[f][x^2,Sqrt[y]],{x,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y]
```

Use a Boolean combination of [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) objects and polynomial inequalities:

```wolfram
CylindricalDecomposition[!g || f&&x y<=1,{x,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y]
```

```wolfram
g=CylindricalDecomposition[ForAll[t,x y z<=t^2 &&t^3<=t x^2+y z+t^4],{x,y,z},"Function"]

CylindricalDecompositionFunction[...][x,y,z]
```

Compute $f⊻g$:


#### Quantifier Elimination

Eliminate quantifiers from quantified [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) objects:

```wolfram
f=CylindricalDecomposition[x y t<=t^3+1 && z<=t z^2+2 x t-y z,{x,y,z,t},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y,z,t]
```

Eliminate quantifiers when the variables are quantified in the same order they appear in the formula:

```wolfram
CylindricalDecomposition[ForAll[z,Exists[t,f]],{x,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y]
```

```wolfram
CylindricalDecomposition[Exists[z,ForAll[t,f]],{x,y},"Function"]
(* Output *)
False
```

```wolfram
CylindricalDecomposition[Exists[{z,t},f],{x,y},"Function"]
(* Output *)
True
```

Eliminate quantifiers from systems involving [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) objects:

```wolfram
f=CylindricalDecomposition[ForAll[t,x y z<=t^2 &&t^3<=x^2+2 x y-z t^4],{x,y,z},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y,z]
```

```wolfram
g=CylindricalDecomposition[x^2<=y z-z^3,{x,y,z},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y,z]
```

```wolfram
CylindricalDecomposition[Exists[z,f&&x y z==1||g&&x^2+y^2>z^2],{x,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y]
```

The cost of quantifier elimination depends on the order of variables:

```wolfram
ineqs=x y z<=z^4-1 && x>z^2-y z;
```

```wolfram
f=CylindricalDecomposition[ineqs,{x,y,z},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y,z]
```

```wolfram
g=CylindricalDecomposition[ineqs,{x,z,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,z,y]
```

When the quantified variables appear in the formula after the free variables, elimination is fast:

```wolfram
(ez1=CylindricalDecomposition[Exists[z,f],{x,y},"Function"])//Timing
(* Output *)
{0.0625,CylindricalDecompositionFunction[...][x,y]}
```

```wolfram
(ey1=CylindricalDecomposition[Exists[y,g],{x,z},"Function"])//Timing
(* Output *)
{0.03125,CylindricalDecompositionFunction[...][x,z]}
```

Elimination of quantifiers from formulas constructed in a different variable order is much slower:

```wolfram
(ez2=CylindricalDecomposition[Exists[z,g],{x,y},"Function"])//Timing
(* Output *)
{0.25,CylindricalDecompositionFunction[...][x,y]}
```

```wolfram
(ey2=CylindricalDecomposition[Exists[y,f],{x,z},"Function"])//Timing
(* Output *)
{2.203125,CylindricalDecompositionFunction[...][x,z]}
```

Check that the obtained results are equivalent:

```wolfram
CylindricalDecomposition[Equivalent[ez1,ez2],{x,y}]
(* Output *)
True
```

```wolfram
CylindricalDecomposition[Equivalent[ey1,ey2],{x,z}]
(* Output *)
True
```

### Properties & Relations

This gives a cylindrical algebraic formula in an encapsulated form:

```wolfram
CylindricalDecomposition[x^2+y^2+z^2<=1,{x,y,z},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y,z]
```

By default, [CylindricalDecomposition](https://reference.wolfram.com/language/ref/CylindricalDecomposition.html) gives the formula written as explicit equations and inequalities:

```wolfram
CylindricalDecomposition[x^2+y^2+z^2<=1,{x,y,z}]
(* Output *)
(x==-1&&y==0&&z==0)||(-1<x<1&&((y==-Sqrt[1-x^2]&&z==-Sqrt[1-x^2-y^2])||(-Sqrt[1-x^2]<y<Sqrt[1-x^2]&&-Sqrt[1-x^2-y^2]<=z<=Sqrt[1-x^2-y^2])||(y==Sqrt[1-x^2]&&z==-Sqrt[1-x^2-y^2])))||(x==1&&y==0&&z==0)
```

[Normal](https://reference.wolfram.com/language/ref/Normal.html) of the [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) object is equivalent, and often identical, to the formula computed directly:

```wolfram
Normal[%%]===%
(* Output *)
True
```

Cylindrical algebraic formulas in encapsulated form are often more efficient for further computation:

```wolfram
qf=∃_ux^2+y^2+z^2+x y z u+u^4==1;
qg=∃_vx+3 y z v^2+v^4<3;
```

```wolfram
{f1,g1}={CylindricalDecomposition[qf,{x,y,z},"Function"],CylindricalDecomposition[qg,{x,y,z},"Function"]};
```

```wolfram
{f2,g2}={CylindricalDecomposition[qf,{x,y,z}],CylindricalDecomposition[qg,{x,y,z}]};
```

Operations on [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) objects are fast:

```wolfram
Timing[CylindricalDecomposition[f1&&g1,{x,y,z},"Function"]]
(* Output *)
{0.609375,CylindricalDecompositionFunction[...][x,y,z]}
```

Operations on cylindrical algebraic formulas written as explicit equations and inequalities often are much slower:

```wolfram
TimeConstrained[CylindricalDecomposition[f2&&g2,{x,y,z}],300]
(* Output *)
$Aborted
```

Use [FindInstance](https://reference.wolfram.com/language/ref/FindInstance.html) to find points that satisfy a [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) object:

```wolfram
CylindricalDecomposition[x^3+y^3>=x y+1,{x,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y]
```

```wolfram
FindInstance[%,{x,y}]
(* Output *)
{{x->-74,y->119}}
```

[SemialgebraicComponentInstances](https://reference.wolfram.com/language/ref/SemialgebraicComponentInstances.html) gives at least one point in each connected component of a semialgebraic set:

```wolfram
SemialgebraicComponentInstances[x^2-y^2>1,{x,y}]
(* Output *)
{{x->-2,y->-1},{x->-2,y->1},{x->2,y->-1},{x->2,y->1},{x->-Sqrt[2],y->0},{x->Sqrt[2],y->0}}
```

[CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) merges several cells to get a more compact representation:

```wolfram
CylindricalDecomposition[x^2-y^2>1,{x,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y]
```

[SemialgebraicComponentInstances](https://reference.wolfram.com/language/ref/SemialgebraicComponentInstances.html) now knows that fewer points suffice:

```wolfram
SemialgebraicComponentInstances[%,{x,y}]
(* Output *)
{{x->-2,y->0},{x->2,y->0}}
```

[GenericCylindricalDecomposition](https://reference.wolfram.com/language/ref/GenericCylindricalDecomposition.html) computes representations of semialgebraic sets set up to lower-dimensional parts:

```wolfram
CylindricalDecomposition[x^2+y^2<=1||x==1,{x,y}]
(* Output *)
(x==-1&&y==-Sqrt[1-x^2])||(-1<x<1&&-Sqrt[1-x^2]<=y<=Sqrt[1-x^2])||x==1
```

```wolfram
GenericCylindricalDecomposition[x^2+y^2<=1||x==1,{x,y}]
(* Output *)
{-1<x<1&&-Sqrt[1-x^2]<y<Sqrt[1-x^2],-1+x==0||-1+x^2+y^2==0}
```

Use [Method](https://reference.wolfram.com/language/ref/Method.html)->{"CylindricalDecompositionFunctionOutput"->[True](https://reference.wolfram.com/language/ref/True.html)} to get [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) results:

```wolfram
GenericCylindricalDecomposition[x^2+y^2<=1||x==1,{x,y},Method->{"CylindricalDecompositionFunctionOutput"->True}]
(* Output *)
{CylindricalDecompositionFunction[...][x,y],-1+x==0||-1+x^2+y^2==0}
```

### Possible Issues

When given inexact arguments, [CylindricalDecompositionFunction](https://reference.wolfram.com/language/ref/CylindricalDecompositionFunction.html) uses the inexact values of arguments for comparisons, but produces an exact result:

```wolfram
CylindricalDecomposition[6x^2+16y^2<5,{x,y},"Function"]
(* Output *)
CylindricalDecompositionFunction[...][x,y]
```

```wolfram
%/.x->0.5
(* Output *)
CylindricalDecompositionFunction[...][y]
```

```wolfram
Normal[%]
(* Output *)
-(Sqrt[(7)/(2)])/(4)<y<(Sqrt[(7)/(2)])/(4)
```


```wolfram
RegionPlot[f,{x,-5,5},{y,-5,5}]
```
