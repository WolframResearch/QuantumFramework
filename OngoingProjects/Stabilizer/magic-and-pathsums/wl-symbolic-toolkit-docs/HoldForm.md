# HoldForm | [SpanFromLeft]

> [HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html)[*expr*] — prints as the expression `*expr*`, with `*expr*` maintained in an unevaluated form.

## Details

[HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html) allows you to see the output form of an expression without performing normal evaluation of  the expression.

[HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html) has attribute [HoldAll](https://reference.wolfram.com/language/ref/HoldAll.html) and performs no operation on its arguments.

[HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html) is removed by [ReleaseHold](https://reference.wolfram.com/language/ref/ReleaseHold.html).

[HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html) is generated by default by [Trace](https://reference.wolfram.com/language/ref/Trace.html) and related functions and can be inserted as a wrapper by such functions as [ToExpression](https://reference.wolfram.com/language/ref/ToExpression.html) and [Extract](https://reference.wolfram.com/language/ref/Extract.html).

Even though `*expr*` itself is not evaluated, [HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html)[*expr*] may still evaluate if `*expr*` is of the form `*f*[*args*]` and upvalues for `*f*` have been defined.

## Examples

### Basic Examples

Addition in held form:

```wolfram
HoldForm[1+1]
(* Output *)
1+1
```

An unevaluated assignment:

```wolfram
HoldForm[x=3]
(* Output *)
x=3
```

An unevaluated power:

```wolfram
HoldForm[34^78]
(* Output *)
34^78
```

Evaluate the expression by applying [ReleaseHold](https://reference.wolfram.com/language/ref/ReleaseHold.html):

```wolfram
ReleaseHold[%]
(* Output *)
285335317058486027411405324166464744463360460002598071329967949465250689557820764970048082035633315075321399848988573696
```

### Scope

Display a sum of squares in unevaluated form:

```wolfram
∑_{i=1}^{10}(HoldForm[#1^2]&)[i]
(* Output *)
1^2+2^2+3^2+4^2+5^2+6^2+7^2+8^2+9^2+10^2
```

View the unevaluated form of an extracted part:

```wolfram
Extract[Hold[{1,1+1,π}], {1,2},HoldForm]
(* Output *)
1+1
```

### Applications

Show a formula and its evaluation:

```wolfram
HoldForm[Integrate[x^2 E^-x^2,x]]==Integrate[x^2 E^-x^2,x]
(* Output *)
∫x^2 ℯ^(-x^2)ⅆx==-(1)/(2) ℯ^(-x^2) x+(1)/(4) Sqrt[π] Erf[x]
```

### Properties & Relations

[HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html) displays without a wrapper:

```wolfram
HoldForm[1+1]
(* Output *)
1+1
```

See the [FullForm](https://reference.wolfram.com/language/ref/FullForm.html) expression:

```wolfram
FullForm[%]
(* Output *)
HoldForm[Plus[1,1]]
```

[Hold](https://reference.wolfram.com/language/ref/Hold.html) displays the held expression with a wrapper:

```wolfram
Hold[1+1]
(* Output *)
Hold[1+1]
```

Use [Evaluate](https://reference.wolfram.com/language/ref/Evaluate.html) to force evaluation of an argument of [HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html):

```wolfram
HoldForm[Evaluate[1+1]]
(* Output *)
2
```

[Sequence](https://reference.wolfram.com/language/ref/Sequence.html) splicing and [Splice](https://reference.wolfram.com/language/ref/Splice.html) still happen inside [HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html):

```wolfram
HoldForm[Sequence[1+1]]
(* Output *)
1+1
```

```wolfram
HoldForm[Splice[{1+1},_]]
(* Output *)
1+1
```

Use [HoldCompleteForm](https://reference.wolfram.com/language/ref/HoldCompleteForm.html) to suppress even such transformations:

```wolfram
HoldCompleteForm[Sequence[1+1]]
(* Output *)
Sequence[1+1]
```

```wolfram
HoldCompleteForm[Splice[{1+1},_]]
(* Output *)
Splice[{1+1},_]
```

Upvalues work inside [HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html):

```wolfram
HoldForm[h[x_]]^:=HoldForm[f[x]]
```

```wolfram
HoldForm[h[1+2]]
(* Output *)
f[1+2]
```

They do not work inside [HoldCompleteForm](https://reference.wolfram.com/language/ref/HoldCompleteForm.html):

```wolfram
HoldCompleteForm[h[x_]]^:=HoldCompleteForm[f[x]]
```

```wolfram
HoldCompleteForm[h[1+2]]
(* Output *)
h[1+2]
```

[ReleaseHold](https://reference.wolfram.com/language/ref/ReleaseHold.html) removes one level of [HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html):

```wolfram
ReleaseHold[HoldForm[1+2]]
(* Output *)
3
```

[HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html) gives an object that is never evaluated:

```wolfram
HoldForm[1+2]
(* Output *)
1+2
```

Copy the output and paste it into an input cell. The `1+2 is still unevaluated:

```wolfram
1+2
(* Output *)
1+2
```

[Defer](https://reference.wolfram.com/language/ref/Defer.html) gives an object whose evaluation is merely deferred until it is explicitly given as Wolfram Language input:

```wolfram
Defer[1+2]
(* Output *)
1+2
```

Copy the output and paste it into an input cell. The `1+2 is evaluated:

```wolfram
1+2
(* Output *)
3
```

[HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html) maintains expressions in unevaluated form, and all parts are inactive:

```wolfram
HoldForm[Sin[ArcTan[1]]]
(* Output *)
Sin[ArcTan[1]]
```

[Inactivate](https://reference.wolfram.com/language/ref/Inactivate.html) maintains symbols in inactive form and allows parts of expressions to be inactive:

```wolfram
Inactive[Sin][ArcTan[1]]
(* Output *)
Sin[(π)/(4)]
```

[HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html) can be used to freeze the result of [ToExpression](https://reference.wolfram.com/language/ref/ToExpression.html) before it is evaluated:

```wolfram
ToExpression["1+1",InputForm,HoldForm]
(* Output *)
1+1
```

### Possible Issues

[HoldPattern](https://reference.wolfram.com/language/ref/HoldPattern.html)[*expr*] is equivalent to `*expr*` for pattern matching but maintains `*expr*` in an unevaluated form:

```wolfram
f[HoldPattern[x_+y_]]:=Hold[x,y]
```

```wolfram
f[Unevaluated[1+2]]
(* Output *)
Hold[1,2]
```

[HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html)[*expr*] is not equivalent to `*expr*` for pattern matching:

```wolfram
g[HoldForm[x_+y_]]:=Hold[x,y]
```

```wolfram
g[Unevaluated[1+2]]
(* Output *)
g[Unevaluated[1+2]]
```

Only a literal [HoldForm](https://reference.wolfram.com/language/ref/HoldForm.html) expression is matched:

```wolfram
g[HoldForm[1+2]]
(* Output *)
Hold[1,2]
```
