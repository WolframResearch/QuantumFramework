# HoldComplete | [SpanFromLeft]

> [HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html)[*expr*] — shields `*expr*` completely from the standard Wolfram Language evaluation process, preventing even upvalues associated with `*expr*` from being used.

## Details

[HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html) allows you to use an expression that has not undergone any evaluation.

[HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html) has attribute [HoldAllComplete](https://reference.wolfram.com/language/ref/HoldAllComplete.html) and performs no operations on its arguments.

[HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html) is removed by [ReleaseHold](https://reference.wolfram.com/language/ref/ReleaseHold.html).

[HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html)[*e*_1,*e*_2,…] maintains a sequence of unevaluated expressions to which a function can be applied using [Apply](https://reference.wolfram.com/language/ref/Apply.html).

[HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html) is generated by default by [MakeExpression](https://reference.wolfram.com/language/ref/MakeExpression.html) and can be inserted as a wrapper by such functions as [ToExpression](https://reference.wolfram.com/language/ref/ToExpression.html) and [Extract](https://reference.wolfram.com/language/ref/Extract.html).

Unlike [Hold](https://reference.wolfram.com/language/ref/Hold.html), [HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html)[*expr*] remains unevaluated even if `*expr*` is of the form `*f*[*args*]` and upvalues for `*f*` have been defined.

## Examples

### Basic Examples

Allow no subexpression to evaluate:

```wolfram
HoldComplete[1+1,Evaluate[1+2],Sequence[3,4]]
(* Output *)
HoldComplete[1+1,Evaluate[1+2],Sequence[3,4]]
```

Evaluate the expression by applying [ReleaseHold](https://reference.wolfram.com/language/ref/ReleaseHold.html):

```wolfram
ReleaseHold[%]
(* Output *)
Sequence[2,3,3,4]
```

### Scope

Extract a part without allowing it to evaluate:

```wolfram
Extract[HoldComplete[{1,1+1,π}], {1,2},HoldComplete]
(* Output *)
HoldComplete[1+1]
```

### Properties & Relations

[HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html) is the standard container with attribute [HoldAllComplete](https://reference.wolfram.com/language/ref/HoldAllComplete.html):

```wolfram
Attributes[HoldComplete]
(* Output *)
{HoldAllComplete,Protected}
```

```wolfram
HoldComplete[Sequence[a,b],1+2]
(* Output *)
HoldComplete[Sequence[a,b],1+2]
```

No form of evaluation control affects an expression with attribute [HoldAllComplete](https://reference.wolfram.com/language/ref/HoldAllComplete.html):

```wolfram
HoldComplete[1+2]
(* Output *)
HoldComplete[1+2]
```

```wolfram
HoldComplete[Evaluate[1+2]]
(* Output *)
HoldComplete[Evaluate[1+2]]
```

```wolfram
HoldComplete[Sequence[a,b]]
(* Output *)
HoldComplete[Sequence[a,b]]
```

```wolfram
g/:HoldComplete[g[x_]]:=x
```

```wolfram
HoldComplete[g[1]]
(* Output *)
HoldComplete[g[1]]
```

Substitution still happens inside [HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html):

```wolfram
HoldComplete[f[1+2]]/.f[x_]:>g[x]
(* Output *)
HoldComplete[g[1+2]]
```

[ReleaseHold](https://reference.wolfram.com/language/ref/ReleaseHold.html) removes one level of [HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html):

```wolfram
ReleaseHold[HoldComplete[Sequence[1,2]]]
(* Output *)
Sequence[1,2]
```

[Hold](https://reference.wolfram.com/language/ref/Hold.html) is a milder form of [HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html) that allows for some form of evaluation:

```wolfram
Hold[1+2]
(* Output *)
Hold[1+2]
```

```wolfram
Hold[Evaluate[1+2]]
(* Output *)
Hold[3]
```

```wolfram
Hold[Sequence[a,b]]
(* Output *)
Hold[a,b]
```

```wolfram
g/:Hold[g[x_]]:=x
```

```wolfram
Hold[g[1]]
(* Output *)
1
```

[HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html) can be used to freeze the result of [ToExpression](https://reference.wolfram.com/language/ref/ToExpression.html) before it is evaluated:

```wolfram
ToExpression["Sequence[a,b]",InputForm,HoldComplete]
(* Output *)
HoldComplete[Sequence[a,b]]
```

[MakeExpression](https://reference.wolfram.com/language/ref/MakeExpression.html) by default wraps the result in [HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html):

```wolfram
MakeExpression["1","+","1",StandardForm]
(* Output *)
HoldComplete[1+1]
```

### Possible Issues

[HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html) affects only evaluation; input transformations are still applied:

```wolfram
FullForm[HoldComplete[a-b,a/b]]
(* Output *)
HoldComplete[Plus[a,Times[-1,b]],Times[a,Power[b,-1]]]
```

[HoldComplete](https://reference.wolfram.com/language/ref/HoldComplete.html) does not prevent formatting:

```wolfram
HoldComplete[Grid[{{1,2},{3,4}}]]
(* Output *)
HoldComplete[{{1, 2}, {3, 4}}]
```

Add [DisableFormatting](https://reference.wolfram.com/language/ref/DisableFormatting.html) to prevent formatting:

```wolfram
HoldComplete[DisableFormatting[Grid[{{1,2},{3,4}}]]]
(* Output *)
HoldComplete[Grid[{{1, 2}, {3, 4}}]]
```
