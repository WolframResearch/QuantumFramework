# IrreduciblePolynomialQ | [SpanFromLeft]

> [IrreduciblePolynomialQ](https://reference.wolfram.com/language/ref/IrreduciblePolynomialQ.html)[*poly*]  — tests whether `*poly*` is an irreducible polynomial over the rationals.
> [IrreduciblePolynomialQ](https://reference.wolfram.com/language/ref/IrreduciblePolynomialQ.html)[*poly*,[Modulus](https://reference.wolfram.com/language/ref/Modulus.html)->*p*] — tests whether `*poly*` is irreducible modulo a prime `*p*`.
> [IrreduciblePolynomialQ](https://reference.wolfram.com/language/ref/IrreduciblePolynomialQ.html)[*poly*,[Extension](https://reference.wolfram.com/language/ref/Extension.html)->{*a*_1,*a*_2,…}] — tests whether `*poly*` is irreducible over the field extension generated by the algebraic numbers `*a*_*i*`.
> [IrreduciblePolynomialQ](https://reference.wolfram.com/language/ref/IrreduciblePolynomialQ.html)[*poly*,[Extension](https://reference.wolfram.com/language/ref/Extension.html)->[All](https://reference.wolfram.com/language/ref/All.html)] — tests whether `*poly*` is absolutely irreducible over the complex numbers.

## Details and Options

The polynomial `*poly*` can involve any number of variables.

[IrreduciblePolynomialQ](https://reference.wolfram.com/language/ref/IrreduciblePolynomialQ.html) takes the following options:

| [Extension](https://reference.wolfram.com/language/ref/Extension.html) | [None](https://reference.wolfram.com/language/ref/None.html) | generators for the algebraic number field to be used |
| --- | --- | --- |
| [GaussianIntegers](https://reference.wolfram.com/language/ref/GaussianIntegers.html) | [False](https://reference.wolfram.com/language/ref/False.html) | whether irreducibility should be tested over Gaussian rationals |
| [Modulus](https://reference.wolfram.com/language/ref/Modulus.html) | 0 | prime modulus to use |

If any coefficients in `*poly*` are complex numbers, irreducibility testing is done over the Gaussian rationals.

With the default setting [Extension](https://reference.wolfram.com/language/ref/Extension.html)->[None](https://reference.wolfram.com/language/ref/None.html), [IrreduciblePolynomialQ](https://reference.wolfram.com/language/ref/IrreduciblePolynomialQ.html)[*poly*] will treat algebraic number coefficients in `*poly*` like independent variables.

[IrreduciblePolynomialQ](https://reference.wolfram.com/language/ref/IrreduciblePolynomialQ.html)[*poly*,[Extension](https://reference.wolfram.com/language/ref/Extension.html)->[Automatic](https://reference.wolfram.com/language/ref/Automatic.html)] extends the domain of coefficients to include any algebraic numbers that appear in `*poly*`.

[IrreduciblePolynomialQ](https://reference.wolfram.com/language/ref/IrreduciblePolynomialQ.html) automatically threads over lists.

## Examples

### Basic Examples

Test irreducibility of polynomials:

```wolfram
IrreduciblePolynomialQ[{x^2-1,x^2-2}]
(* Output *)
{False,True}
```

### Scope

Irreducibility of univariate polynomials over the rationals:

```wolfram
IrreduciblePolynomialQ[{x^2+1,x^3-8}]
(* Output *)
{True,False}
```

Irreducibility of multivariate polynomials over the rationals:

```wolfram
IrreduciblePolynomialQ[{x^4-4y^2,x^4-2y^2}]
(* Output *)
{False,True}
```

Irreducibility over the Gaussian rationals:

```wolfram
IrreduciblePolynomialQ[x^2+2 I x -1]
(* Output *)
False
```

```wolfram
IrreduciblePolynomialQ[x^2+1,GaussianIntegers->True]
(* Output *)
False
```

Irreducibility of univariate polynomials over the integers modulo `2:

```wolfram
IrreduciblePolynomialQ[{x^3+1,x^3+x+1},Modulus->2]
(* Output *)
{False,True}
```

Irreducibility of multivariate polynomials over the integers modulo `3:

```wolfram
IrreduciblePolynomialQ[{x y+x+1,2+2 x+2 x^2+2 y+x y},Modulus->3]
(* Output *)
{True,False}
```

Irreducibility over a finite field:

```wolfram
ℱ=FiniteField[17,3];
```

```wolfram
IrreduciblePolynomialQ[{ℱ[3]x^2+ℱ[1771],ℱ[3]x^2+ℱ[1771]y^3}]
(* Output *)
{False,True}
```

```wolfram
IrreduciblePolynomialQ[{x^3+5x+19,x^4+9x+3},Extension->ℱ]
(* Output *)
{False,True}
```

By default, algebraic number coefficients are treated as independent variables:

```wolfram
IrreduciblePolynomialQ[x^2+2Sqrt[2]x+2]
(* Output *)
True
```

This tests irreducibility over the rationals extended by the algebraic number coefficients:

```wolfram
IrreduciblePolynomialQ[x^2+2Sqrt[2]x+2,Extension->Automatic]
(* Output *)
False
```

Irreducibility over a finite algebraic extension of rationals:

```wolfram
IrreduciblePolynomialQ[{x^3-2,x^3-3},Extension->2^(1/3)]
(* Output *)
{False,True}
```

```wolfram
IrreduciblePolynomialQ[{x^2-2y^4,x^4-3 y^2},Extension->Sqrt[3]]
(* Output *)
{True,False}
```

Absolute irreducibility over the complex numbers:

```wolfram
IrreduciblePolynomialQ[{x^3-3,x^2+2 x y-7},Extension->All]
(* Output *)
{False,True}
```

### Options

#### Extension

By default, algebraic number coefficients are treated as independent variables:

```wolfram
IrreduciblePolynomialQ[x^2+2Sqrt[3]x y+3y^2]
(* Output *)
True
```

[Extension](https://reference.wolfram.com/language/ref/Extension.html)->[Automatic](https://reference.wolfram.com/language/ref/Automatic.html) automatically extends to a field that covers the coefficients:

```wolfram
IrreduciblePolynomialQ[x^2+2Sqrt[3]x y+3y^2,Extension->Automatic]
(* Output *)
False
```

The polynomial $x^{2}+2$ is irreducible over the rationals:

```wolfram
IrreduciblePolynomialQ[x^2+2]
(* Output *)
True
```

The same polynomial is reducible over the rationals extended by [I](https://reference.wolfram.com/language/ref/I.html) and [Sqrt](https://reference.wolfram.com/language/ref/Sqrt.html)[2]:

```wolfram
IrreduciblePolynomialQ[x^2+2,Extension->{I,Sqrt[2]}]
(* Output *)
False
```

Absolute irreducibility:

```wolfram
IrreduciblePolynomialQ[x^7+12x y-11,Extension->All]
(* Output *)
True
```

The polynomial $x^{3}+5 x+19$ is irreducible over $\mathbb{Z}_{17}$:

```wolfram
IrreduciblePolynomialQ[x^3+5x+19,Modulus->17]
(* Output *)
True
```

The same polynomial is reducible over a degree-$3$ extension of $\mathbb{Z}_{17}$:

```wolfram
IrreduciblePolynomialQ[x^3+5x+19,Extension->FiniteField[17,3]]
(* Output *)
False
```

The polynomial $f$ is irreducible over the finite field $\mathcal{F}$ with $29^{3}$ elements:

```wolfram
ℱ=FiniteField[29,3];
f=ℱ[12]x^2+ℱ[34]x+ℱ[56]
(* Output *)
![image](img/image_001.png)
```

```wolfram
IrreduciblePolynomialQ[f]
(* Output *)
True
```

$f$ becomes reducible after embedding $\mathcal{F}$ in a finite field $\mathcal{G}$ with $29^{6}$ elements:

```wolfram
𝒢=FiniteField[29,6];
ℰ=FiniteFieldEmbedding[ℱ,𝒢]
(* Output *)
FiniteFieldEmbedding[<|interpretation -> FiniteFieldElement[FiniteField[29, 27, +, 2, #, +, #, ^, 3, &, Polynomial], 01], index -> 29, shortIndex -> 29, indexShortened -> True, characteristic -> 29, shortCharacteristic -> 29, extensionDegree -> 3, field -> FiniteField[...], fieldDisplayed -> False|>-><|interpretation -> FiniteFieldElement[FiniteField[29, 2, +, 13, #, +, 17, #, ^, 2, +, 25, #, ^, 3, +, #, ^, 4, +, #, ^, 6, &, Polynomial], 32217165], index -> 114060686, shortIndex -> 114060686, indexShortened -> True, characteristic -> 29, shortCharacteristic -> 29, extensionDegree -> 6, field -> FiniteField[...], fieldDisplayed -> False|>]
```

```wolfram
IrreduciblePolynomialQ[f,Extension->ℰ]
(* Output *)
False
```

#### GaussianIntegers

The polynomial $x^{2}+1$ is irreducible over the rationals:

```wolfram
IrreduciblePolynomialQ[x^2+1]
(* Output *)
True
```

The same polynomial is reducible over the Gaussian rationals:

```wolfram
IrreduciblePolynomialQ[x^2+1,GaussianIntegers->True]
(* Output *)
False
```

#### Modulus

Irreducibility modulo a prime:

```wolfram
IrreduciblePolynomialQ[{x^2+2,x^2+3},Modulus->17]
(* Output *)
{False,True}
```

### Properties & Relations

A polynomial is irreducible if [FactorList](https://reference.wolfram.com/language/ref/FactorList.html) gives one nonconstant factor with exponent 1:

```wolfram
{IrreduciblePolynomialQ[x^2+4],FactorList[x^2+4]}
(* Output *)
{True,{{1,1},{4+x^2,1}}}
```

```wolfram
{IrreduciblePolynomialQ[x^2-4],FactorList[x^2-4]}
(* Output *)
{False,{{1,1},{-2+x,1},{2+x,1}}}
```
